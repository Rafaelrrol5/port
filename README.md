# Portfolio Pixel Dark

Portfolio pessoal com visual escuro em pixel art, feito com HTML, CSS e JavaScript.

## Paginas

- `index.html`: pagina inicial do portfolio
- `sobre.html`: pagina de apresentacao pessoal
- `projetos.html`: pagina com projetos atuais e futuros

## Observacao sobre o projeto JoJo

A pagina `projetos.html` possui um botao que aponta para `jojo.html`.
Se voce ja tem `jojo.html` no seu repositorio, mantenha esse arquivo junto com o novo portfolio para o link funcionar.

## Como publicar

Envie os arquivos para um repositorio do GitHub e ative o GitHub Pages na branch principal.
